# Komendy do wklejenia - merge i push

# 1. Dodaj zaktualizowany README.md
git add README.md

# 2. Zrób merge z remote (bez rebase)
git pull --no-rebase origin main

# 3. Jeśli są konflikty, rozwiąż je i wykonaj:
# git add .
# git commit

# 4. Dodaj wszystkie pozostałe pliki
git add .

# 5. Commit wszystkich zmian
git commit -m "feat: v2.0 - Complete update with RAG, history, export, and UI improvements

- Add backend infrastructure with Express.js
- Implement RAG system for legal document verification
- Add analysis modes (Information/Problem)
- Add history sidebar with search
- Add PDF and DOCX export functionality
- Add print optimization
- Refactor to custom hooks (useAnalysis, useExport, useHistory)
- Add sticky navigation tabs
- Improve tooltips with React Portal
- Update README.md for v2.0 (remove banner, add changelog)"

# 6. Push
git push origin main
